# Exergame for Stroke Rehabilitation using Deep Learning-Based Hand Gesture Recognition

* by **Luthfi**
* Undergraduate Thesis
* Extended research based on **Sulistio (2025)**

---

## Purpose

This project aims to develop an **exergame for stroke rehabilitation** that utilizes **hand gesture recognition** based on deep learning models (GRU).
The game is designed to support **home-based rehabilitation** by providing interactive physical exercises through gesture-controlled gameplay.

Research objectives:

1. Undergraduate Thesis development
2. Extension and improvement of prior research by **Sulistio (2025)**
3. Evaluation of deep learning models for hand gesture-based rehabilitation games

---

## Research Context

This project is part of an **experimental study** comparing deep learning models for hand gesture recognition in an exergame environment.
Multiple training iterations and parameter variations are applied to handle the **fluctuative nature of deep learning outputs**.
